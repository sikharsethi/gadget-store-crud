# Gadget Store - CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) web application built using Spring Boot.

## Technologies Used

- Spring Boot
- Spring MVC
- MySQL
- Thymeleaf
- Maven
- HTML / CSS

## Features

- Add new gadgets
- View all gadgets
- Update gadget details
- Delete gadgets

## Project Structure

- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer
- Thymeleaf Templates

## Database

Database used: MySQL

Example Table:
Product
- id
- name
- brand
- price

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Configure MySQL database in `application.properties`
4. Run `GadgetStoreApplication.java`
5. Open browser and go to:http://localhost:8080
